# 🧠 AI Hub — Лаунчер для ИИ-чатов

**AI Hub** — кроссплатформенное десктопное приложение на **pywebview + HTML/CSS/JS**, объединяющее популярные ИИ-чаты в одном интерфейсе.

![Версия](https://img.shields.io/badge/версия-5.6.0-blue)
![Python](https://img.shields.io/badge/python-3.8+-green)
![Технология](https://img.shields.io/badge/технология-pywebview+HTML/CSS/JS-purple)
![Лицензия](https://img.shields.io/badge/лицензия-MIT-green)

---

## ✨ Возможности

- 🎨 **Чистый HTML/CSS/JS** — без CustomTkinter и Kivy
- 🍎 **macOS стиль** — кнопки управления в стиле Mac (справа)
- 🌐 **WebView внутри приложения** — чаты открываются в отдельных окнах
- 🚀 **Быстрый запуск** — мгновенное открытие любого ИИ-чата
- 🪟 **Frameless окна** — без системных рамок
- 🎯 **Без браузера** — нет адресной строки, панелей, devtools
- ⚡ **240 Гц анимации** — плавные переходы с elastic-кривой Безье
- 🔄 **Автопроверка обновлений** — проверка версии через GitHub API
- 💻 **Qwen Code CLI** — автоматическая установка и запуск через npm/winget/choco
- 📦 **Модульная архитектура** — разделение на независимые модули
- 🔲 **Кнопка выхода из fullscreen** — быстрое сворачивание из полноэкранного режима
- 🌊 **Эффект Liquid Glass** — динамические градиенты, полупрозрачные слои с размытием, блики при движении курсора
- 🎭 **Переключение тем** — 🌓 Светлая / Тёмная / Системная + кастомные акцентные цвета

## 📁 Структура проекта

```
AI Hub/
├── main.py              # Точка входа, API, конфигурация
├── proverka.py          # Проверка обновлений (GitHub API)
├── qwen.py              # Установка и запуск Qwen Code CLI
├── index.py             # HTML генератор
├── requirements.txt     # Зависимости
├── README.md            # Документация
├── build.py             # Скрипт сборки
├── compile.bat          # BAT-файл для сборки
└── AI_Hub.spec          # Spec-файл PyInstaller
```

### Модули:

| Файл | Описание |
|------|----------|
| `main.py` | Главный файл, инициализация приложения, API для JS |
| `proverka.py` | Класс `VersionChecker` — проверка обновлений через GitHub |
| `qwen.py` | Класс `QwenCodeInstaller` — установка и запуск Qwen Code |
| `index.py` | Генерация HTML интерфейса (меню и окна чатов) |

## 🔄 Проверка обновлений

При запуске приложение автоматически проверяет наличие новой версии через GitHub API.

**Настройка:** В `main.py` укажите ваш репозиторий:
```python
GITHUB_REPO = "your-username/ai-hub"  # ЗАМЕНИТЕ на ваш репозиторий!
```

- ✅ **Установлена актуальная версия** — в консоли появится зелёное сообщение
- 🎉 **Доступна новая версия** — в консоли появится уведомление с датой релиза и списком изменений

## 💻 Qwen Code CLI

При клике на карточку **Qwen Code**:

1. Приложение проверяет наличие `npm`
2. Если `npm` не найден → пробует `winget`
3. Если `winget` не найден → пробует `choco`
4. Устанавливает: `npm install -g @qwen-code/qwen-code@0.12.6`
5. **Автоматически запускает** `qwen` после установки

**Что происходит:**
- Открывается новое CMD окно с заголовком "Qwen Code"
- Выполняется установка
- После успешной установки выводится сообщение и запускается `qwen`
- Окно остаётся открытым для работы

## 🐛 Решение проблем

### Окно не открывается
Проверьте, что все файлы на месте:
- `main.py`
- `proverka.py`
- `qwen.py`
- `index.py`

### Кнопки не работают
Убедитесь, что pywebview установлен корректно:
```bash
pip install --upgrade pywebview
```

### Чат не загружается
Проверьте подключение к интернету

### Qwen Code не устанавливается
Установите один из менеджеров пакетов:
- [Node.js (npm)](https://nodejs.org/)
- [winget](https://learn.microsoft.com/ru-ru/windows/package-manager/winget/)
- [Chocolatey](https://chocolatey.org/)

### CMD окно не открывается при установке Qwen Code
Убедитесь, что у приложения есть права на открытие новых процессов. Запустите от имени администратора.
